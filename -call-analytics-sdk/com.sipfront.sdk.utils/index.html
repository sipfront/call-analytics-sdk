@Serializable(with = MaskedStringSerializer::class)
class MaskedString(data: String, index: Int = DEFAULT_INDEX)

Implementation of String that will mask its data with MASK char after defined index when toString is called, while still maintaining data during JSON serialization/deserialization. Useful for preventing leaking of sensitive data like credentials when logging objects that contain these.
